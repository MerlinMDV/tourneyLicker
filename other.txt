    }

commands:

/start - opens form to create new tourney (admin)
/end - ends tourney and distributes rewards (admin)
/status - view current reward recipients and start date (admin)
/leaderboard - view leaderboard (user)
/points add <amount> <user> - add <amount> points to <user> (admin)
/points remove <amount> <user> - remove <amount> points from <user> (admin)
/points view <user> - view points of user
